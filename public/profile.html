Assistant Architecture

The site uses a lightweight retrieval-augmented generation approach. Questions are routed through an approved source corpus, relevant sections are retrieved and scored, and answers are constrained to source-backed material.

  • Frontend: Vite, vanilla JavaScript, HTML, CSS, Tailwind CSS, custom CSS.
  • Hosting: Firebase Hosting.
  • Backend: Firebase Cloud Functions, Node.js 22.
  • LLM integration: Claude API through a backend synthesize function.
  • Retrieval: custom source corpus, intent routing, scoring, and source-bound answer generation.
  • QA: Vitest, Vite build checks, scripted retrieval verification, and browser inspection.
